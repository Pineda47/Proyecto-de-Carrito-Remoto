En esta etapa se analiza si las interacciones entre nodos están funcionando. Para ello, se comienza con el primer nodo, que es el de `ultra_sonido`, mencionado en la rama de desarrollo previo.  

Se realiza la creación del código, el cual se enfoca en generar valores aleatorios de tipo `int32` hasta alcanzar un total de 10.
